1. Ensure to you can securely access your github account from terminal. This may require ssl setup.
2. Create public repo in your github account.
3. Copy URL to new repo.
4. Go to terminal and 'git clone <repo>'
5. CD to new <repo> directory.
6. rm -rf .git directory

7. NOTE : C++ NOT REQUIRED FOR CAPSTONE PROJECT

    Open Clion, create new C++ Executable Project NOT from existing sources BUT IN <repo> directory.
    This will create a CMakeLists.txt and main.cpp "Hello, World!" app that you can build and run to
    confirm project setup.

8.1. NOTES :
    * REQUIRES PYTHON 2.7 AND INSTALL REQUIREMENTS.TXT.
    * DO NOT USE VIRTUALENV OR MINICONDA.
    * DO NOT USE "sudo apt autoremove". SEE https://carnd.slack.com/archives/C2J6AVC5V/p1506234519000039

    Open Pycharm, create new Python project, adding existing source files just cloned above.

8.2. DON'T DO THIS :
    File -> Settings -> Project -> Project Interpreter -> Settings 'cog' -> Add local -
> "/home/andrew/miniconda3/envs/tensorflow/bin/python"

8.3. DO THIS :
    File -> Settings -> Project -> Project Structure -> + Add Content Root ->
    check that is cloned home directory, like "/home/andrew/dev/term3/sdc-term3-p2-semantic-segmentation", else
    add and 'x' to delete previously shown Content Root.
